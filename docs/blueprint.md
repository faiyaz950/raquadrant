# **App Name**: RaQuadrant Energy Solutions

## Core Features:

- Homepage with Intro: Display key services, intro to company vision, mission, and contact details.
- About Us Page: Detail company's vision, mission, expertise and client-centric approach.
- Services and Solutions Page: Describe services (rooftop solar, pumps, lighting), scope of work (system selection, approvals, support), and project lifecycle.
- Contact Form: Allow users to submit inquiries with a contact form that sends details to info@raquadrantenergy.com.
- Interactive Project Map: Visually display geographic presence across Eastern UP, Bihar, Jharkhand, etc. Allow users to explore past projects.

## Style Guidelines:

- Primary color: Light orange (#FFB347) to convey energy and warmth.
- Background color: Off-white (#FFFFFF), for a clean and modern feel.
- Accent color: A muted, analogous yellow-orange (#FFC857) for buttons and highlights.
- Body and headline font: 'PT Sans' for a modern but friendly, easily-readable look.
- Use clean, professional icons to represent different services and solutions.
- Maintain a clean and modern layout with clear sections and easy navigation.
- Subtle transitions and animations to enhance user experience.
